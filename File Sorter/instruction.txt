In order to run the Text File Compression task, the following software packages must be installed on the system:

- **zip utility**: Required for compressing `.txt` files into a `.zip` archive.
  - **Linux (Ubuntu/Debian)**: `sudo apt-get install zip`
  - **Linux (Arch/Manjaro)**: `sudo pacman -S zip`

- **Python**: Required to run the Python implementation of the task.
  - **Installation**: `pip install python`
  
These dependencies ensure that the program can create `.zip` files and run without errors.
